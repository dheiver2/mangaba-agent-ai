Capacidades

Um operador completo, não um chatbot.

Loop ReAct de verdade: pensa, age com ferramentas, observa e corrige — com as técnicas de context engineering do Manus embutidas.

👁️

Enxerga

Analisa imagens, screenshots, gráficos e documentos fotografados com describe_image (visão local).

🎙️

Ouve e fala

Transcreve reuniões com Whisper local, responde mensagens de voz e conversa pelo modo voz.py.

🌐

Navega

Pesquisa na web, abre o navegador de verdade (clica, preenche, extrai) e busca páginas como markdown com fetch_url.

📊

Gera documentos reais

Planilhas .xlsx, propostas .docx, apresentações .pptx, PDFs e gráficos — arquivos de verdade, não só texto.

📄

Lê contratos e notas

Extrai texto e tabelas de PDFs com read_pdf — contratos, notas fiscais, relatórios.

💻

Programa e executa

Python e shell nativos, edição de arquivos, e extensão ilimitada via servidores MCP.

Modos de execução

Do comando rápido à operação agendada.

Cada tamanho de tarefa tem seu modo — incluindo estratégias contra janelas de contexto pequenas.

01

Direto

Uma tarefa, um comando.

python main.py --prompt "..."
02

Voz

Fale a tarefa; ele executa e responde falando.

python voz.py
03

Pipeline

Tarefas grandes em fases com contexto limpo.

python pipeline.py --prompt "..."
04

Fila

Processamento em série + cron pra rotinas.

python fila.py add "..." && python fila.py run
05

Janela gigante

GitHub Models grátis: até 1M tokens de contexto.

python main.py --model openai/gpt-4.1-mini
06

Servidor MCP

Vire ferramenta de qualquer cliente MCP.

python run_mcp_server.py

Planejamento · Reflexão · Human-in-the-loop
